Statistical Check of Inter-rater Reliability: Krippendorff's Alpha



How many data sets do you have?*

This should be the number of questions, raters, or other items that you want to check reliability for. For example, if you are checking inter-rater reliability amongst esasy raters and there are 5 raters, please insert "5". If you want to check the reliability of 5 human raters, please enter "5", etc. This program currently only supports between 2 and 10 items.



What type of data do you have?*

Select "categorical" if raters answered a category (i.e., "yes/no", "green/blue/red"), select "ordinal" if raters answered a Likert-scale question or rated on a subjective scale, select "continuous" if the measurements were made on a true scale (e.g., time, etc.)
