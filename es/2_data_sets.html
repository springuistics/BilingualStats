Statistical Comparison of Two Numerical Data Sets

Answer the following questions about your data sets:

Are your data sets paired data?*

You should select "yes" if your two data sets consist of the same participants. For example, select "yes" if you are using a pre-/post- design, or if you are comparing two different survey questions answered by the same group of participants.

Are your data sets both continuous data?*

You should select "yes" if BOTH of your data sets are measured on scales in which: a) many different numerical answers are possible, and b) a score higher by "1" is equally opposite to a score lower by "1". For example, data measured in time, percentages, number of times, and number of correct answers are often considered to be continuous scales. Likert-scale survey questions are NOT considered to be continuous, so if either of your data sets are of this type, you should select "no." If you are unsure, you can select "yes," and the program will automatically check the data for normalcy via a Shapiro-Wilks Test. If either sets of your data fail this test, it will automatically be considered ordinal and not continuous.
